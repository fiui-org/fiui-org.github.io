## 如何更新

修改后，push到master分支即可。提交更改之前请先在本地预览以确定没有错误。

### python

如果本地有python环境，在项目根目录执行

```bash
$ python -m SimpleHTTPServer 8001
```

来开启一个web服务，然后访问[http://localhost:8001](http://localhost:8001)

### php

如果本地有php环境，在项目根目录执行

```bash
$ php -S 0.0.0.0:8001
```
